# Xin chào, tôi là Đắc Đoàn 👋

Tôi là sinh viên ngành Công nghệ đa phương tiện, định hướng phát triển theo vị trí IT Staff / IT Helpdesk / Web Admin trong doanh nghiệp và khu công nghiệp.

## Kỹ năng
- Thiết kế và quản trị website
- HTML, CSS, JavaScript cơ bản
- Thiết kế giao diện bằng Figma
- Quản trị nội dung website
- Hỗ trợ người dùng, cài đặt phần mềm, xử lý lỗi máy tính cơ bản
- Kiểm tra mạng LAN, máy in, thiết bị văn phòng

## Dự án cá nhân
### Website bán giày online
- Xây dựng giao diện website bán hàng
- Có trang quản trị admin
- Quản lý sản phẩm, đơn hàng, khách hàng
- Thiết kế giao diện dễ sử dụng, phù hợp bán hàng online

## Mục tiêu nghề nghiệp
Tôi mong muốn làm việc ở vị trí IT Staff / IT Helpdesk để hỗ trợ hệ thống máy tính, mạng nội bộ, website và các thiết bị công nghệ trong doanh nghiệp.
